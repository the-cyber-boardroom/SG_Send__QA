What is SG/Send?

SG/Send is a secure, encrypted file sharing platform that enables controlled distribution of sensitive documents. Key properties:

  • Zero-knowledge encryption — files are encrypted in the browser before upload; the server never sees plaintext
  • Token-based access gate — each sharing session is scoped to a unique access token
  • Key-in-URL — the decryption key lives in the URL fragment, never sent to the server
  • Audit trail — all access attempts (including invalid tokens) are logged

What Is This Site?

This documentation is automatically generated from Playwright browser tests that run against the live SG/Send platform on every CI push. Every page documents a real user workflow, captured as screenshots and verified by automated assertions.

The pipeline: tests run → screenshots are captured → pages are generated → site is deployed. No manual writing. No stale documentation. If a test fails, the page is not updated.
