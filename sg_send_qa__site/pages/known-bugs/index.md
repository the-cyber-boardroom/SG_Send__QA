# Known Bugs

Tests in this section are marked `xfail` — they **run on every CI pass** and are expected to fail. If they unexpectedly pass, CI will flag it so the fix can be confirmed and the bug entry retired.

Each page shows screenshots captured during the failing run so the exact broken behaviour is documented.
